A modern WordPress plugin boilerplate with React/TypeScript admin UI, REST API infrastructure, and modular PHP architecture.

== Description ==

**WPAB Boilerplate** is a modern starting point for building WordPress plugins. It bridges the gap between classic WordPress development and modern JavaScript tooling.

**Key Features:**

*   **Modern Stack**: React, TypeScript, Tailwind CSS, and Webpack.
*   **Modular PHP**: Namespace-based autoloader, Singleton pattern, and clean architecture.
*   **REST API Ready**: Base controller class with permission checks and sample endpoints.
*   **Database Manager**: Easy custom table creation on activation.
*   **Settings API**: Integrated settings management with schema validation.
*   **White Labeling**: Fully filterable branding (names, icons, URLs).

**Developer Friendly:**

*   Pre-configured **ESLint**, **Prettier**, and **TypeScript**.
*   **GitHub Actions** workflow included (optional).
*   **Dual Build System**: Supports modern and legacy browser builds.

== Installation ==

1.  Clone the repository or download the zip.
2.  Run `npm install` to install dependencies.
3.  Run `npm run build` to compile assets.
4.  Activate the plugin in WordPress.
